# 🗳️ חמ"ל בחירות חכם (Election Command Center)

מערכת ניהול בחירות מתקדמת ומאובטחת שנועדה לרכז, לנהל ולבצע אופטימיזציה של תהליך הבחירות בשטח. המערכת מאפשרת למטה ניהול מרכזי לשלוט בנתוני הבוחרים, לעקוב אחר אחוזי הצבעה בזמן אמת, לנהל מנהלי שטח, ולבצע פעולות ממוקדות כמו חיוג מהיר ותזכורות אישיות לבוחרים.

---

## 🛠️ טכנולוגיות (Tech Stack)
המערכת נבנתה באמצעות טכנולוגיות מודרניות מבוססות ענן:

* **Frontend**: HTML5, CSS3, JavaScript (ES6+).
* **Database**: Google Firebase Firestore (מסד נתונים בזמן אמת).
* **Visualization**: Chart.js (להצגת גרפיקה ונתוני הצבעה).
* **Data Parsing**: PapaParse (לעיבוד קבצי CSV של רשימות בוחרים).
* **Hosting**: Firebase Hosting.

---

## ✨ תכונות מרכזיות
* **ניהול בוחרים דינמי:** טעינת מאגרי בוחרים מ-CSV וניהול סטטוס הצבעה אישי.
* **שליטה מבוססת הרשאות:** הפרדה בין מנהל מערכת (Admin) בעל הרשאות מלאות לבין אחראי שטח (User) הרואה רק את הבוחרים שלו.
* **דשבורד חכם:** מדדי התקדמות ויזואליים (Progress bars) וגרפים של התפלגות הצבעה.
* **כלים ממוקדים בשטח:** חיוג מהיר (Click-to-Call), ניהול תזכורות אישיות, ושליחת בקשות הסעה דרך WhatsApp.
* **אבטחה:** מנגנון כניסה מבוסס קוד גישה מוגן.
